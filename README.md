# 鼓動の刃

## 概要
心拍数を制御して時間を操り、敵を倒す2Dアクションゲームです。

## 開発環境
- **言語**: C++ / HLSL / GLSL
- **フレームワーク**: Siv3D
- **IDE**: Visual Studio 2022
- **バージョン管理**: Git / GitHub

## 担当ソースコード
本プロジェクトにおいて、主に以下の機能の実装を担当しました。

- **`Player.cpp` / `Player.h`**
  - プレイヤーの移動、ジャンプ、攻撃などの挙動制御。
  - Stateパターンを用いた状態遷移の管理。
- **`AllEffect.cpp` / `AllEffect.h`**
  - 衝突や攻撃演出を管理する汎用エフェクトシステム。
- **`GameUI.cpp` / `GameUI.h`**
  - 心拍数やHPなど、ゲーム情報の可視化を行うUI表示処理。

## アピールポイント

### 1. 軽量で汎用的なエフェクトシステム
管理クラスを通すことで、ゲーム内の演出の追加・差し替えが容易な設計にしています。`AllEffect` クラスにより、描画処理の分離と効率化を図りました。

### 2. Stateパターンを用いた状態管理
複雑になりがちなプレイヤーのアクションをStateパターンで実装しました。状態遷移を厳密に管理することで、意図しない挙動を防ぎ、拡張性の高い堅牢なコード構造を実現しています。

## 操作方法
- **移動**: Aキー（左） / Dキー（右）
- **ジャンプ**: Wキー
- **攻撃**: マウス左クリック
- **時間操作**: マウス右クリック（※心拍数が一定値以下の時のみ発動可能）
