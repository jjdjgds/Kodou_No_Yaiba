# 鼓動の刃

## 概要
心拍数を制御して時間を操り、敵を倒す2Dアクションゲームです。

## 開発環境
- **言語**: C++ / HLSL / GLSL
- **フレームワーク**: Siv3D
- **IDE**: Visual Studio 2022
- **バージョン管理**: Git / GitHub

## アピールポイント

### 1. 軽量で汎用的なエフェクトシステム
ゲーム内の衝突や攻撃などの演出を効率的に管理できるエフェクトシステムを設計しました。管理クラスを通すことで、演出の追加や差し替えが容易な設計にしています。
- **実装参照**: `AllEffect.h`

### 2. Stateパターンを用いた状態管理
プレイヤーの複雑なアクション（移動、ジャンプ、攻撃など）をStateパターンで実装しました。状態遷移を厳密に管理することで、意図しない挙動やバグを未然に防ぐ、堅牢な構造を意識しています。
- **実装参照**: `Player.cpp`

## 操作方法
- **移動**: Aキー（左） / Dキー（右）
- **ジャンプ**: Wキー
- **攻撃**: マウス左クリック
- **時間操作**: マウス右クリック（※心拍数が一定値以下の時のみ発動可能）
